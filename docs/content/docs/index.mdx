## What is Agent Factory?

Agent Factory is a TypeScript CLI tool for running Codex against Linear-tracked work inside isolated sandbox environments. It manages the full lifecycle of syncing code into a sandbox, invoking the agent, and syncing changes back.

## Key Concepts

- **Sandbox** -- An isolated environment (Docker container or local directory) where an agent executes code
- **Host** -- Your machine where Agent Factory runs and the real git repo lives
- **Agent** -- The AI coding tool invoked inside the sandbox (e.g. Codex)
- **Iteration** -- A single invocation of the agent, producing at most one commit

## Quick Start

### 1. Install

```bash
npm install -g @imsthegenius/agent-factory
```

### 2. Initialize

Run `factory init` in your repository to scaffold the config directory:

```bash
factory init
```

This creates a `.factory/` directory with a Dockerfile, prompt file, and configuration.

### 3. Run

```bash
factory
```

Agent Factory will build the Docker image, sync your code into the sandbox, and start Codex.
